Реквізити єдиного депозитного рахунку для оплати митних платежів

Для стягнення митних платежів за товари які переміщуються через митну територію України, митні органи резервують на своєму рахунку для подальшого списування грошові кошти платника, для зарахування коштів попередньої оплати за здійснення митного оформлення. Для розрахунку митних платежів можете скористатися калькулятором розрахунку митних платежів, для більш точного розрахунку суми митних платежів та отримання консультації надішліть заявку нашим фахівцям. Починаючи з 01.01.2020 року зарахування коштів авансових платежів (передоплати) підприємствами здійснюється виключно на новий депозитний рахунок 3557, відкритий у Державній казначейській службі України на ім’я Державної митної служби України.
